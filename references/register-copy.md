# Register: marketing vs blog vs formál

To, čo je slop v jednom registri, môže byť správne v inom. Pred passom urči register textu a podľa neho posudzuj.

## Marketingový / webový copy (landing, služby, hero)

**Cieľ:** konkrétny benefit, dôvera, akcia. Krátke vety, oslovenie čitateľa, jasné CTA.

**Tu je slop najtvrdší:**
- Prázdne prídavné mená bez dôkazu (`špičkový`, `komplexný`) sú zabijak dôvery.
- Clichés ("individuálny prístup", "spoľahlivý partner") = okamžitá strata kredibility.
- Generická všeobecnosť namiesto čísel a konkrétností.

**Čo NIE je slop tu:** priame oslovenie ("ozvite sa"), výzva na akciu, krátke úderné vety, emócia ak je úprimná. Nepenalizuj copy za to, že je copy (to je chyba encyklopedického anchora).

## Blog / článok

**Cieľ:** informovať, presvedčiť, ukázať expertízu. Dlhší dych, príklady, dáta, názor.

**Tu je slop:**
- Filler otvorenia ("V dnešnej dobe").
- Šablónové nadpisy ("Prečo je X dôležité").
- "Na záver" zhrnutia čo opakujú.
- Rule of three v každom odseku.
- Žiadne konkrétne čísla/mená/zdroje.

**Čo NIE je slop:** osobná skúsenosť, vyhranený názor, odbočka ak je zaujímavá, dlhšia veta ak nesie myšlienku.

## Formálny / oficiálny text (zmluvy, oznámenia, úrady)

**Cieľ:** presnosť, jednoznačnosť. Tu je vyšší register a istá miera "suchosti" namieste.

**Tu je slop:**
- Anglicizmy a kalky ("adresovať", "na dennej báze").
- Falošná dôležitosť ("je dôležité poznamenať").
- Buzzwordy ("synergia", "holistický").

**Čo NIE je slop:** odborná terminológia, pasíva ak je nutná pre neosobnosť normy, dlhšie súvetia ak sú presné. Neprepisuj zákonný text na marketingový.

## Rýchla tabuľka

| Jav | Marketing | Blog | Formál |
|---|---|---|---|
| priame oslovenie "vy" | OK | OK striedmo | nie |
| krátke úderné vety | OK | OK | striedmo |
| výzva na akciu | OK | OK | nie |
| pasíva | nie | striedmo | OK ak nutná |
| osobný názor | striedmo | OK | nie |
| odborný žargón | striedmo | striedmo | OK |

**Pravidlo:** identifikuj register PRED rezaním. Marketingový copy nie je pokazený blog a formálny text nie je pokazený marketing.
